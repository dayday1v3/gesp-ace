# GESP Ace 后端设计文档

> 版本：v1.0 | 创建日期：2026-05-09

## 1. 项目概述

### 1.1 后端定位

GESP Ace 后端是面向青少儿编程考级的 AI 智能陪练工具的服务端实现，提供题目管理、用户学习进度跟踪、编程题判题、AI 诊断等核心能力。

### 1.2 技术选型

| 技术 | 版本 | 说明 |
|-----|------|------|
| **Node.js** | 20.x | 运行时环境 |
| **Express** | 4.x | Web 框架 |
| **TypeScript** | 5.x | 类型安全 |
| **Prisma** | 5.x | ORM 框架 |
| **PostgreSQL** | 16.x | 关系型数据库 |
| **Redis** | 7.x | 缓存数据库 |
| **Judge0 CE** | Latest | 代码判题引擎 |
| **JWT** | - | 身份认证 |
| **Docker** | 24.x | 容器化部署 |

### 3.2 数据表设计

#### 3.2.1 用户表 (users)

```sql
CREATE TABLE users (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    email VARCHAR(100) UNIQUE,
    avatar VARCHAR(500) DEFAULT '/avatars/default.png',
    current_level INTEGER DEFAULT 1,
    total_score INTEGER DEFAULT 0,
    streak_days INTEGER DEFAULT 0,
    total_practice_days INTEGER DEFAULT 0,
    exam_date DATE,
    last_practice_date DATE,
    role VARCHAR(20) DEFAULT 'student',
    status VARCHAR(20) DEFAULT 'active',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_users_username ON users(username);
CREATE INDEX idx_users_email ON users(email);
```

CREATE TABLE levels (

#### 3.2.4 题目表 (questions)

```sql
CREATE TABLE questions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    type VARCHAR(20) NOT NULL,
    content TEXT NOT NULL,
    options JSONB,
    answer TEXT NOT NULL,
    analysis TEXT,
    difficulty INTEGER DEFAULT 1 CHECK (difficulty BETWEEN 1 AND 3),
    points INTEGER DEFAULT 2,
    level_id INTEGER REFERENCES levels(id),
    knowledge_point_id INTEGER REFERENCES knowledge_points(id),
    year INTEGER,
    season VARCHAR(10),
    source VARCHAR(50),
    test_cases JSONB,
    time_limit INTEGER DEFAULT 1000,
    memory_limit INTEGER DEFAULT 256,
    status VARCHAR(20) DEFAULT 'active',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_questions_level ON questions(level_id);
CREATE INDEX idx_questions_knowledge ON questions(knowledge_point_id);
CREATE INDEX idx_questions_type ON questions(type);
CREATE INDEX idx_questions_difficulty ON questions(difficulty);
```

#### 3.2.5 练习记录表 (practices)

```sql
CREATE TABLE practices (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id),
    type VARCHAR(30) NOT NULL,
    status VARCHAR(20) DEFAULT 'in_progress',
    target_level INTEGER,
    target_knowledge VARCHAR(20),
    started_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    completed_at TIMESTAMP,
    total_score INTEGER DEFAULT 0,
    correct_count INTEGER DEFAULT 0,
    wrong_count INTEGER DEFAULT 0,
    time_spent INTEGER DEFAULT 0
);

CREATE INDEX idx_practices_user ON practices(user_id);
CREATE INDEX idx_practices_type ON practices(type);
CREATE INDEX idx_practices_started ON practices(started_at);
```

#### 3.2.6 答题记录表 (answers)

```sql
CREATE TABLE answers (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    practice_id UUID NOT NULL REFERENCES practices(id),
    question_id UUID NOT NULL REFERENCES questions(id),
    user_answer TEXT,
    is_correct BOOLEAN DEFAULT FALSE,
    score INTEGER DEFAULT 0,
    time_spent INTEGER DEFAULT 0,
    submitted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_answers_practice ON answers(practice_id);
CREATE INDEX idx_answers_question ON answers(question_id);
```

#### 3.2.7 错题本表 (mistakes)

```sql
CREATE TABLE mistakes (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID NOT NULL REFERENCES users(id),
    question_id UUID NOT NULL REFERENCES questions(id),
    error_type VARCHAR(50),
    error_count INTEGER DEFAULT 1,
    last_reviewed_at TIMESTAMP,
    mastered BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(user_id, question_id)
);

CREATE INDEX idx_mistakes_user ON mistakes(user_id);
CREATE INDEX idx_mistakes_mastered ON mistakes(user_id, mastered);
```

#### 3.2.8 收藏表 (favorites)

```sql
CREATE TABLE favorites (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

CREATE INDEX idx_favorites_user ON favorites(user_id);
```

## 4. API 接口设计

### 4.1 API 规范

- **Base URL**: `/api/v1`
- **认证方式**: JWT Bearer Token
- **请求格式**: JSON
- **响应格式**:
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

### 4.2 认证模块

// src/services/judgeService.ts
interface JudgeResult {
  accepted: boolean;
  score: number;
  output: string;
  error?: string;
  testCases: TestCaseResult[];
  time: number;
  memory: number;
}

interface TestCaseResult {
  passed: boolean;
  expectedOutput: string;
  actualOutput: string;
  time: number;
  memory: number;
  score: number;
}

class JudgeService {
  async submitCode(code: string, questionId: string): Promise<JudgeResult> {
    const question = await this.getQuestion(questionId);
    const testCases = question.testCases;
    
    const results: TestCaseResult[] = [];
    let totalScore = 0;
    
    for (const testCase of testCases) {
      const result = await this.runSingleTest(code, testCase);
      results.push(result);
      if (result.passed) {
        totalScore += testCase.score || (question.points / testCases.length);
      }
    }
    
    return {
      accepted: results.every(r => r.passed),
      score: Math.round(totalScore),
      output: results[0]?.actualOutput || '',
      testCases: results,
      time: Math.max(...results.map(r => r.time)),
      memory: Math.max(...results.map(r => r.memory))
    };
  }
  
  private async runSingleTest(code: string, testCase: TestCase): Promise<TestCaseResult> {
    // 调用 Judge0 API
    const response = await fetch(`${JUDGE0_URL}/submissions?base64=0&wait=true`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        source_code: Buffer.from(code).toString('base64'),
        language_id: 54, // C++17
        stdin: Buffer.from(testCase.input).toString('base64'),
        expected_output: Buffer.from(testCase.output).toString('base64'),
        cpu_time_limit: 1,
        memory_limit: 256000
      })
    });
    
    const result = await response.json();
    return {
      passed: result.status?.id === 3, // 3 = accepted
      expectedOutput: testCase.output,
      actualOutput: result.stdout,
      time: result.time,
      memory: result.memory,
      score: result.status?.id === 3 ? testCase.score || 5 : 0
    };
  }
}

### 5.2 AI 诊断服务 (DiagnosisService)

```typescript
// src/services/diagnosisService.ts
interface WeakPoint {
  knowledgeCode: string;
  knowledgeName: string;
  errorRate: number;
  recentMistakes: number;
  trend: 'improving' | 'stable' | 'worsening';
}

interface Recommendation {
  type: 'daily' | 'topic' | 'exam';
  title: string;
  description: string;
  knowledgeCodes: string[];
  priority: number;
}

class DiagnosisService {
  async analyzeUserPerformance(userId: string): Promise<{
    weakPoints: WeakPoint[];
    recommendations: Recommendation[];
    predictedScore: { low: number; mid: number; high: number };
  }> {
    // 1. 获取用户最近答题数据
    const recentAnswers = await this.getRecentAnswers(userId, 30);
    
    // 2. 按知识点统计错误率
    const knowledgeStats = this.calculateKnowledgeStats(recentAnswers);
    
    // 3. 识别薄弱知识点
    const weakPoints = this.identifyWeakPoints(knowledgeStats);
    
    // 4. 生成学习建议
    const recommendations = this.generateRecommendations(weakPoints);
    
    // 5. 预测分数
    const predictedScore = this.predictScore(knowledgeStats);
    
    return { weakPoints, recommendations, predictedScore };
  }
  
  private calculateKnowledgeStats(answers: Answer[]): Map<string, KnowledgeStat> {
    const stats = new Map();
    
    for (const answer of answers) {
      const kp = answer.question.knowledgePoint;
      if (!stats.has(kp.code)) {
        stats.set(kp.code, { 
          total: 0, 
          correct: 0,
          recentCorrect: 0,
          recentTotal: 0
        });
      }
      
      const stat = stats.get(kp.code);
      stat.total++;
      if (answer.isCorrect) stat.correct++;
      
      // 最近 7 天的数据
      if (this.isRecent(answer.submittedAt, 7)) {
        stat.recentTotal++;
        if (answer.isCorrect) stat.recentCorrect++;
      }
    }
    
    return stats;
  }
  
  private identifyWeakPoints(stats: Map<string, KnowledgeStat>): WeakPoint[] {
    const weakPoints: WeakPoint[] = [];
    
    for (const [code, stat] of stats) {
      const errorRate = 1 - (stat.correct / stat.total);
      const recentErrorRate = stat.recentTotal > 0 
        ? 1 - (stat.recentCorrect / stat.recentTotal) 
        : errorRate;
      
      // 错误率 > 30% 或最近错误率上升
      if (errorRate > 0.3 || recentErrorRate > errorRate) {
        const knowledge = this.getKnowledgePoint(code);
        weakPoints.push({
          knowledgeCode: code,
          knowledgeName: knowledge.name,
          errorRate: Math.round(errorRate * 100) / 100,
          recentMistakes: stat.recentTotal - stat.recentCorrect,
          trend: this.determineTrend(errorRate, recentErrorRate)
        });
      }
    }
    
    return weakPoints.sort((a, b) => b.errorRate - a.errorRate);
  }
  
  private predictScore(stats: Map<string, KnowledgeStat>): { low: number; mid: number; high: number } {
    let totalWeight = 0;
    let weightedCorrect = 0;
    
    for (const [_, stat] of stats) {
      const weight = stat.total;
      const correctRate = stat.correct / stat.total;
      weightedCorrect += weight * correctRate;
      totalWeight += weight;
    }
    
    const baseRate = totalWeight > 0 ? weightedCorrect / totalWeight : 0.6;
    
    return {
      low: Math.round(baseRate * 80),
      mid: Math.round(baseRate * 90),
      high: Math.round(Math.min(100, baseRate * 100))
    };
  }
}
```

// src/services/practiceService.ts
class PracticeService {
  async startDailyPractice(userId: string): Promise<Practice> {
    // 检查今日是否已完成
    const todayCompleted = await this.checkDailyCompleted(userId);
    if (todayCompleted) {
      throw new Error('今日已完成');
    }
    
    // 获取每日一练题目（选择×1 + 判断×1 + 编程×1）
    const questions = await this.questionService.getDailyQuestions({
      types: ['choice', 'judgment', 'coding'],
      count: 3,
      userId
    });
    
    // 创建练习记录
    const practice = await this.prisma.practices.create({
      data: {
        userId,
        type: 'daily',
        status: 'in_progress',
        startedAt: new Date()
      }
    });
    
    // 关联题目
    for (const question of questions) {
      await this.prisma.answers.create({
        data: {
          practiceId: practice.id,
          questionId: question.id
        }
      });
    }
    
    return { ...practice, questions };
  }
  
  async submitAnswer(userId: string, practiceId: string, questionId: string, answer: string): Promise<AnswerResult> {
    const practice = await this.prisma.practices.findUnique({
      where: { id: practiceId },
      include: { answers: true }
    });
    
    if (practice.userId !== userId) {
      throw new Error('Unauthorized');
    }
    
    const question = await this.questionService.getQuestion(questionId);
    const isCorrect = this.checkAnswer(question, answer);
    
    // 更新答题记录
    const answerRecord = practice.answers.find(a => a.questionId === questionId);
    await this.prisma.answers.update({
      where: { id: answerRecord.id },
      data: {
        userAnswer: answer,
        isCorrect,
        score: isCorrect ? question.points : 0
      }
    });
    
    // 如果错误，加入错题本
    if (!isCorrect) {
      await this.addToMistakes(userId, question);
    }
    
    return {
      isCorrect,
      correctAnswer: question.answer,
      analysis: question.analysis,
      score: isCorrect ? question.points : 0
    };
  }
  
  async completePractice(userId: string, practiceId: string): Promise<CompleteResult> {
    const practice = await this.prisma.practices.findUnique({
      where: { id: practiceId },
      include: { answers: true }
    });
    
    const totalScore = practice.answers.reduce((sum, a) => sum + a.score, 0);
    const correctCount = practice.answers.filter(a => a.isCorrect).length;
    
    // 更新练习记录
    await this.prisma.practices.update({
      where: { id: practiceId },
      data: {
        status: 'completed',
        completedAt: new Date(),
        totalScore,
        correctCount,
        wrongCount: practice.answers.length - correctCount
      }
    });
    
    // 检查连续打卡
    const { streakDays, isNewStreak } = await this.updateStreak(userId);
    
    // 检查成就
    const newAchievements = await this.checkAchievements(userId, practice);
    
    // 更新等级进度
    await this.updateLevelProgress(userId, practice);
    
    return {
      totalScore,
      correctCount,
      wrongCount: practice.answers.length - correctCount,
      streakDays,
      newAchievements
    };
  }
}

class AchievementService {
  async checkAndAwardAchievements(userId: string, context: AchievementContext): Promise<Achievement[]> {
    const user = await this.getUser(userId);
    const earnedIds = user.achievements.map(a => a.id);
    const allAchievements = await this.getAllAchievements();
    
    const newAchievements: Achievement[] = [];
    
    for (const achievement of allAchievements) {
      if (earnedIds.includes(achievement.id)) continue;
      
      const earned = await this.checkCondition(user, achievement, context);
      if (earned) {
        await this.awardAchievement(userId, achievement);
        newAchievements.push(achievement);
      }
    }
    
    return newAchievements;
  }
  
  private async checkCondition(user: User, achievement: Achievement, context: AchievementContext): Promise<boolean> {
    switch (achievement.conditionType) {
      case 'practice_count':
        return context.totalQuestions >= achievement.conditionValue;
      
      case 'streak_days':
        return user.streakDays >= achievement.conditionValue;
      
      case 'level_mastered':
        const progress = await this.getLevelProgress(user.id, achievement.conditionValue);
        return progress.status === 'mastered';
      
      case 'daily_perfect':
        return context.dailyPerfect === true;
      
      default:
        return false;
    }
  }
}

export const authMiddleware = async (req: Request, res: Response, next: NextFunction) => {
  const token = req.headers.authorization?.replace('Bearer ', '');
  
  if (!token) {
    return res.status(401).json({
      code: 401,
      message: '未登录'
    });
  }
  
  try {
    const decoded = jwt.verify(token, JWT_SECRET) as TokenPayload;
    const user = await prisma.users.findUnique({
      where: { id: decoded.userId },
      select: { id: true, username: true, role: true }
    });
    
    if (!user) {
      return res.status(401).json({
        code: 401,
        message: '用户不存在'
      });
    }
    
    req.user = user;
    next();
  } catch (error) {
    return res.status(401).json({
      code: 401,
      message: 'Token 无效'
    });
  }
};

// src/middlewares/errorHandler.ts
export const errorHandler = (err: Error, req: Request, res: Response, next: NextFunction) => {
  console.error('Error:', err);
  
  if (err instanceof ValidationError) {
    return res.status(400).json({
      code: 400,
      message: err.message,
      errors: err.errors
    });
  }
  
  if (err instanceof UnauthorizedError) {
    return res.status(401).json({
      code: 401,
      message: err.message
    });
  }
  
  if (err instanceof ForbiddenError) {
    return res.status(403).json({
      code: 403,
      message: err.message
    });
  }
  
  if (err instanceof NotFoundError) {
    return res.status(404).json({
      code: 404,
      message: err.message
    });
  }
  
  res.status(500).json({
    code: 500,
    message: '服务器内部错误'
  });
};

## 10. 安全性设计

### 10.1 密码安全

- 密码使用 bcrypt 加密，cost factor = 12
- 登录失败锁定：5次失败后锁定15分钟
- 密码强度要求：至少8位，包含字母和数字

### 10.2 JWT 安全

- Access Token 有效期：2小时
- Refresh Token 有效期：7天
- Token 存储在 HttpOnly Cookie

### 10.3 API 安全

- 所有 API 强制 HTTPS
- 请求体大小限制：1MB
- 文件上传限制：5MB
- SQL 注入防护：参数化查询
- XSS 防护：输入输出转义

### A. 前端 API 调用示例

```typescript
// src/services/api.ts
import axios from 'axios';

const api = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL,
  timeout: 30000
});

api.interceptors.request.use((config) => {
  const token = localStorage.getItem('token');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

api.interceptors.response.use(
  (response) => response.data,
  (error) => {
    if (error.response?.status === 401) {
      localStorage.removeItem('token');
      window.location.href = '/login';
    }
    return Promise.reject(error);
  }
);

export const authAPI = {
  login: (data: { username: string; password: string }) =>
    api.post('/auth/login', data),
  register: (data: { username: string; password: string; email?: string }) =>
    api.post('/auth/register', data)
};

export const questionAPI = {
  getDaily: () => api.get('/questions/daily'),
  getList: (params: QuestionQuery) => api.get('/questions', { params }),
  getDetail: (id: string) => api.get(`/questions/${id}`)
};

export const practiceAPI = {
  start: (data: StartPracticeDTO) => api.post('/practice/start', data),
  submitAnswer: (data: SubmitAnswerDTO) => api.post('/practice/answer', data),
  complete: (practiceId: string) => api.post('/practice/complete', { practiceId })
};

export const codeAPI = {
  run: (data: { code: string; questionId: string }) => api.post('/code/run', data),
  submit: (data: { code: string; questionId: string; practiceId: string }) =>
    api.post('/code/submit', data)
};
```
