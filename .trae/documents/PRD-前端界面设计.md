# GESP Ace 前端开发 PRD 文档

> 版本：v1.0 | 创建日期：2026-05-08

---

## 1. 产品概述

GESP Ace 是一款面向青少儿（9-15岁）编程考级的AI智能陪练工具前端界面。通过"每日一练+智能诊断+个性化学习路径"帮助考生高效备考GESP C++ 1-8级。界面设计追求温暖友好、活泼有趣，避免冰冷的科技感，让学习像游戏一样有趣。

---

## 2. 设计理念

### 2.1 核心设计哲学："温暖的代码伙伴"

**设计关键词**：
- 🎨 **温暖友好** - 像大哥哥大姐姐在指导学习
- 🌈 **活力满满** - 充满趣味和惊喜
- 💫 **即时满足** - 每一步都有正反馈
- 🏠 **轻松自在** - 没有压力，像在自己的房间学习

### 2.2 避免的AI味元素

| ❌ 避免使用 | ✅ 改用 |
|-----------|--------|
| 紫色/蓝色渐变背景 | 温暖的橙色/粉色/黄色系 |
| 深色科技风界面 | 明亮温暖的浅色主题 |
| 几何网格/粒子背景 | 柔和的云朵/星星装饰 |
| 冷冰冰的系统字体 | 活泼可爱的圆润字体 |
| 过于正式的卡片设计 | 圆角气泡/手绘风边框 |
| 机械化的动画 | 有弹性的、俏皮的动效 |

### 3.5 动画规范

**动画原则**：
- 🎯 **有弹性** - 使用 bounce/elastic 缓动函数
- ⚡ **快速响应** - 交互反馈 150-300ms
- 🎭 **有性格** - 不同元素有不同"性格"的动画

```css
--transition-fast: 150ms cubic-bezier(0.4, 0, 0.2, 1);
--transition-base: 300ms cubic-bezier(0.4, 0, 0.2, 1);
--transition-slow: 500ms cubic-bezier(0.4, 0, 0.2, 1);
--transition-bounce: 600ms cubic-bezier(0.68, -0.55, 0.265, 1.55);
```

**核心动画场景**：
1. **页面加载**：卡片从下往上浮入，stagger间隔100ms
2. **按钮交互**：按下时缩小95%，释放时弹回
3. **答题正确**：绿色光晕 + 打勾动画 + 轻微抖动庆祝
4. **答题错误**：红色闪烁 + 轻微摇晃
5. **打卡成功**：星星飘落 + 徽章弹出
6. **等级提升**：金光闪烁 + 升级动画

## 5. 组件设计

### 5.1 等级卡片组件

**状态**：
- 🟢 **已掌握**：绿色边框 + 星星图标
- 🟡 **进行中**：黄色边框 + 进度百分比
- 🔴 **待加强**：红色边框 + 感叹号
- 🔒 **已锁定**：灰色 + 锁图标 + 模糊效果

**交互**：
- Hover：上浮8px + 阴影加深
- Click：缩小95% + 弹回
- 解锁动画：金色光芒 + 星星散落

### 5.2 题目卡片组件

**选择题型**：
- 默认：白色背景 + 浅灰边框

- 选中：橙色背景 + 白色文字
- 正确：绿色背景 + 打勾
- 错误：红色背景 + 显示正确答案

### 5.3 按钮组件

**主要按钮**：
```css
background: linear-gradient(135deg, #FF8C42, #FFB074);
border-radius: 50px;
box-shadow: 0 4px 15px rgba(255, 140, 66, 0.4);
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
```

**交互**：
- Hover：亮度提升10% + 阴影加深
- Active：缩小95% + 阴影减小
- Loading：脉冲动画

### 5.4 徽章/成就组件

- 圆形或盾形设计
- 金色边框 + 渐变背景

- 圆角胶囊形状

### 6.1 页面过渡

**前进动画**：
```css
.page-enter {
  opacity: 0;
  transform: translateY(30px);

.page-enter-active {
  opacity: 1;
  transform: translateY(0);
  transition: all 400ms cubic-bezier(0.34, 1.56, 0.64, 1);
```

**返回动画**：
```css
.page-exit {
  opacity: 1;
  transform: translateX(0);

.page-exit-active {
  opacity: 0;
  transform: translateX(-30px);
  transition: all 300ms ease-in;
```

**正确**：
```
✓ 打勾图标从0放大到1.2再到1
```

## 7. 响应式设计

### 7.1 断点定义

```css
/* 移动端优先 */
--breakpoint-sm: 640px;   /* 大手机 */
--breakpoint-md: 768px;   /* 平板 */
--breakpoint-lg: 1024px;  /* 小笔记本 */
--breakpoint-xl: 1280px;  /* 桌面 */
```

### 7.2 布局适配

**桌面（≥1024px）**：
- 最多4列等级卡片
- 侧边导航栏
- 宽敞的间距

**平板（768-1024px）**：
- 最多3列等级卡片
- 顶部导航栏
- 适中间距

**手机（<768px）**：
- 2列等级卡片
- 底部导航栏
- 紧凑布局
- 触摸友好的大按钮
